## 📋 Projects Overview

### 🎙️ **Audio Transcription AI**
**Location:** `/speech_to_text/`

An intelligent real-time audio transcription application powered by OpenAI Whisper.

**✨ Key Features:**
- 🎤 **Real-time audio recording** with live status indicators
- 🌍 **Multi-language support** (10+ languages including Arabic, French, English)
- 🎯 **High-accuracy transcription** using OpenAI Whisper models
- 📱 **Modern responsive UI** with gradient animations
- ⏰ **Timestamp support** for detailed transcription analysis
- 🔄 **Automatic audio conversion** for optimal processing
- 🎛️ **Multiple audio device support** with device selection

**🎯 Use Cases:**
- Meeting transcriptions
- Interview recordings
- Lecture notes
- Voice memos
- Content creation
- Language learning

---

### 🏢 **Business Name Checker AI**
**Location:** `/Dénominateur/`

An intelligent business name validation system for Tunisia's RNE (Registre National des Entreprises) with comprehensive AI-powered analysis.

**✨ Key Features:**
- 🔍 **Smart name extraction** from conversational queries
- 🛡️ **Multi-layer validation** (special characters, inappropriate content, similarity)
- 📊 **Real-time database checking** against existing business names
- 🤖 **AI-powered chat interface** with contextual assistance
- 🌍 **Multi-language support** (Arabic, French, English)
- 💡 **Intelligent name suggestions** when conflicts are found
- 📈 **Similarity scoring** with detailed conflict analysis
- 🔄 **Alternative name generation** with cultural awareness

**🎯 Use Cases:**
- Business registration preparation
- Trademark conflict avoidance
- Brand name validation
- Legal compliance checking
- Startup name brainstorming
- Corporate rebranding

**🔣 Validation Rules:**
- ✅ Letters, numbers, spaces, basic punctuation (`. - ' &`)
- ❌ Special characters (`@ # $ % ^ * ( ) + = [ ] { } | \ : ; " < > ? / ~`)
- ❌ Inappropriate content detection
- ❌ Existing name conflicts

## 🚀 **Quick Start Guide**

### **For Audio Transcription:**
```bash
cd speech_to_text/
streamlit run app.py
```

1. 🎤 Click "Start Recording"
2. 🗣️ Speak into your microphone
3. ⏹️ Click "Stop Recording"  
4. 🚀 Click "Transcribe" and get results!

### **For Business Name Checking:**
```bash
cd business-checker/
streamlit run app.py
```

1. 💬 Ask: *"Is TechCorp available?"*
2. 🔍 Get instant validation results
3. 💡 Receive alternative suggestions if needed
4. ✅ Proceed with confidence!
